# JSON Serialization & Deserialization Tests

This project contains simple examples of JSON serialization and deserialization across three different programming languages: JavaScript, Python, and C#. It serves as a baseline for comparing how different languages and libraries handle JSON data.

## Project Structure

- `javascript/`: Implementation using built-in `JSON` object.
- `python/`: Implementation using built-in `json` module.
- `csharp/`: Implementation using .NET and `Newtonsoft.Json`.
